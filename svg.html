SVG

1) Scalable Vector Graphics (SVG) are images that look good even when they're scaled to different sizes. Because SVGs are created using XML, they can be easily manipulated. The cirle on the left was drawn using inline svg.

2) The logo added to the upper left corner of this site is an svg. Originally, the image was 512px by 512px. To resize it, I changed the height and width attribute within the svg file. The file size is virtually unchanged regardless of what size the height and width are set(what size the image displays). And it can be resized as many times as needed without any degradation of the image.

3) The new shopping bag image on the upper right of the "Kutz" page is also an svg. It has an id and two classes assigned inside the file. As the previous bag was an icon, I assigned it an id, but now I can apply CSS using the id already within the file. With the logo, I changed the height and width within the file, but on the shopping bag, I've adjusted the size using CSS on the shopping bag's internal id.

4) I can also apply CSS to both of the shopping bag's classes(cls-1 and cls-2). In this case, I'm changing the color of the classes to fit with the color palette of my site.

5) SVGs are incredibly easy to animate using CSS animations. Now when a customer is on the "Kutz" page and adds an item to the bag, they get feedback that their click was received in the form of bag movement.
